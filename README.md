# Fighting Game Notes

I wanted to move away from using OneNote to store most of my notes since the phone app is clunky and its integration with OneDrive is annoying, and I want to be able to link things I've recorded to others.

This GitHub Pages repo leverages [Colin Wilson's Lotus Docs](https://github.com/colinwilson/lotusdocs) theme for the static site generator, Hugo.

All of these are my notes, so take them with a grain of salt.
