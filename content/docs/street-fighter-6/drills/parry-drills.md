FChamp came up with a series of drills to getting the most out of perfect parry in common situations. We're stuck with this mechanic so learn to use it.

All of these trials will have different perfect parry goals, but they should all be done three times in a row. No fudging, if you miss one after getting two, you start from the beginning anyway.

## Parrying staggered lights or target combos

Set opposing character to Marisa
Record jab tc xx gladius on block

Goal: parry the second hit of the tc

## Parrying crossups and post-crossup button pressure

Set opposing character to JP
Record j.lk crossup > cr.lk > cr.lp > cr.lp

First goal: parry the crossup itself
Second goal: parry the first cr.lp

## Parrying staggered mediums and fireball enders

Set opposing character to Juri, infinite stocks
Record drive rush st.mp > st.mp > cr.mp > fireball

First goal: parry the cr.mp
Second goal: parry the fireball (really hard!)

## Parrying midscreen fireballs

Set opposing character to Ryu
Record:
- lp fireball -> lp fireball
- lp fireball -> hp fireball
- hp fireball -> hp fireball

First recording: perfect parry the first fireball, attempt jump OS (holding upforward will net jumpin punish if they're doing consecutive fireballs)
Second and Third recording: parry the second fireball

When parrying these fireballs, avoid blocking. Stand or walk forward and parry. You will take damage both in training and in game, but perfect parrying after crouchblocking still pushes you back.

## Parrying advancing specials

Set opposing character to Honda
Record:
- hp headbutt
- mp headbutt
- hk buttslam
- lk buttslam

Goal: parry each special 3 times in a row

## Parrying Luke Sandblast 

Sandblast is so specific it needs its own parry drill

Set opposing character to Luke
Record:
- mp sandblast
- hp sandblast

Goal: parry each special 3 times in a row
