## Recognize DI and cancel to super

Take any character:

1. set dummy to block all
2. set block reversal slot 1 to "do nothing"
3. set block reversal slot 2 to "drive impact"

Press a medium/heavy frametrap button (some characters like Chun have to do medium/medium) on dummy. If dummy DIs you, cancel your second button into any super.

Do this 5 times on each side.

## Capitalize off drive rush jab

Take any character:

1. set dummy to random block
2. set dummy to always counter hit when struck
3. place dummy in corner

Do drive rush 2lp/5lp:

- On block, tick throw
- On hit, convert into a medium

Do this 6 times on each side. 

Optionally, practice linking the light into a heavy since you'll have both drive rush and counter hit advantage. Light > Medium confirms are easier and more consistent although the damage reward is lower.

## React to whiffed counterpoke after poking

Take any character, but Ken is easier to start with:

1. set dummy to block all
2. set block reversal slots 1, 4, and 5 to "do nothing"
3. set block reversal slot 2 to 2mk
4. set block reversal slot 3 to 5hp

Dummy can be placed anywhere.

Press either 5lk, 5lp, or 2mp to prompt a response:

- On block, do nothing
- On hit, cancel into super or meterless bnb

Do this 5 times on each side.

Optionally, instead of cancelling into super do a drive rush cancel.

## React to jumpin success

Take any character:

1. set dummy to random block
2. set block reversal slot 1 to normal throw with 6f delay
3. set block reversal slots 2 and 3 to "do nothing"

Dummy can be placed anywhere.

Jumpin with jump hp:

- On block, shimmy > 5hp
- On hit, medium starter combo

Do this 5 times on each side.

Optionally, this can be made easier or harder by using lights or heavies as your combo starter on hit.

## React to strike

Take any character and set to random block.

Press 5hp or other cancellable single hit confirm button:

- On hit, DRC
- On block, do nothing and restart

Do this 8 out of 10 times.
